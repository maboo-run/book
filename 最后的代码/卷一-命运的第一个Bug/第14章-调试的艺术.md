def find_max(numbers):
    if not numbers:
        return None
    
    max_num = numbers[0]  # 用第一个元素初始化
    for num in numbers[1:]:  # 从第二个开始比较
        if num > max_num:
            max_num = num
    return max_num
```

```

或者：

```python
def remove_duplicates(lst):
    result = []
    for item in lst:
        if item not in result:
            result.append(item)
    return result
```

Bug1：无限循环

```python
def countdown(n):
    while n > 0:
        print(n)

```

```python
def countdown(n):

        print(n)
        n -= 1

```

        greeting = f"你好, {name}"

Bug3：类型错误

```python
def double_numbers(nums):
    result = []
    for num in nums:
        result.append(num * 2)
    return result

# 测试
numbers = "12345"  # Bug: 传入字符串而不是列表
result = double_numbers(numbers)
print(result)
```

运行输出：

"字符串也能遍历，但`*2`变成重复字符串了！"林北恍然大悟。

```python
def double_numbers(nums):
    if not isinstance(nums, list):
        raise TypeError("参数必须是列表")
    
    result = []
    for num in nums:
        result.append(num * 2)
    return result
```

Bug4：逻辑错误

```python
def is_leap_year(year):
    # Bug: 逻辑不完整
    if year % 4 == 0:
        return True
    return False

print(is_leap_year(2000))  # True，正确
print(is_leap_year(1900))  # True，错误！1900不是闰年
```

"闰年规则：能被4整除，但整百年必须能被400整除。"苏晴说。

修复：

```python
def is_leap_year(year):
    if year % 400 == 0:
        return True
    if year % 100 == 0:
        return False
    if year % 4 == 0:
        return True
    return False
```

Bug5：列表引用问题

```python
def create_matrix(rows, cols):
    row = [0] * cols
    matrix = [row] * rows  # Bug: 浅拷贝
    return matrix

m = create_matrix(3, 3)
m[0][0] = 1
print(m)
```

输出：

```
[[1, 0, 0], [1, 0, 0], [1, 0, 0]]
```

"修改一个，全都变了！"王胖子惊呼。

"因为三行引用的是同一个列表对象。"苏晴解释。

修复：

```python
def create_matrix(rows, cols):
    matrix = []
    for i in range(rows):
        row = [0] * cols
        matrix.append(row)
    return matrix
```

或用列表推导式：

```python
def create_matrix(rows, cols):
    return [[0] * cols for _ in range(rows)]
```
