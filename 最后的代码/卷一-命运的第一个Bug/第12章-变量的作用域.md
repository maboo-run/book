"什么是作用域?"马老师指着房子的图,"简单说,就是变量的'势力范围'。"

"你们看这个房子。客厅、卧室、厨房……每个房间都是一个独立的空间。"

"在卧室里的东西,客厅里拿不到。在厨房里的刀,卧室里用不了。"

"Python的作用域有四种层级,简称LEGB规则:"

马老师在白板上写下:

```
L - Local(局部作用域): 函数内部
E - Enclosing(嵌套作用域): 外层函数
G - Global(全局作用域): 模块级别
B - Built-in(内置作用域): Python内置的
```

"遇到一个变量,Python会按照这个顺序查找:先找局部,再找外层,再找全局,最后找内置。"

王胖子举手:"老师,能举个例子吗?我有点懵。"

"好。"马老师在屏幕上敲代码:

```python
x = "全局变量"

def outer():
    x = "外层函数变量"
    
    def inner():
        x = "内层函数变量"
        print(f"inner中的x: {x}")
    
    inner()
    print(f"outer中的x: {x}")

outer()
print(f"全局的x: {x}")
```

运行:

```
inner中的x: 内层函数变量
outer中的x: 外层函数变量
全局的x: 全局变量
```

"看,每个函数都有自己的`x`,互不干扰。"

"这就像三个同名的人住在不同房间,各过各的日子。"

```python
def player_attack(score):
    return score + 10

def collect_coin(score):
    return score + 5

def take_damage(score, damage_percent):

    return score - damage

# 使用
score = 0
score = player_attack(score)
print(f"攻击后: {score}")

score = collect_coin(score)
print(f"捡金币后: {score}")

score = player_attack(score)
print(f"再次攻击后: {score}")

score = take_damage(score, 0.5)
print(f"受伤后: {score}")
```

```
捡金币后: 15
再次攻击后: 25
受伤后: 12.5
```

"有时候,你有嵌套函数,内层函数想修改外层函数的变量。"

```python
def outer():
    count = 0
    
    def inner():
        nonlocal count  # 声明使用外层函数的变量
        count += 1
        print(f"inner被调用了{count}次")
    
    inner()
    inner()
    inner()
    print(f"outer中的count: {count}")

outer()
```

运行:

```

inner被调用了2次
inner被调用了3次
outer中的count: 3
```

"看,`nonlocal`让内层函数能修改外层函数的变量。"

"但同样的,非必要不使用。能用参数返回值解决的,就别用`nonlocal`。"

```python
import random

def play_once():
    """
    玩一次打地鼠
    返回: True表示打中,False表示未打中
    """
    # 随机出现地鼠
    gopher_position = random.randint(1, 3)
    
    print(f"地鼠出现了!在哪个洞?(1/2/3)")
    player_choice = int(input("你的选择: "))
    
    if player_choice == gopher_position:
        print("打中了!")
        return True
    else:
        print(f"没打中!地鼠在{gopher_position}号洞")
        return False

def calculate_score(hits):
    """
    计算分数
    参数: hits - 命中次数
    返回: 分数
    """
    return hits * 10

def calculate_hit_rate(hits, total):
    """
    计算命中率
    参数: hits - 命中次数, total - 总次数
    返回: 命中率(百分比)
    """
    return (hits / total) * 100

def main():
    """主程序"""
    print("=== 打地鼠游戏 ===")
    print("你有10次机会,每次打中得10分!")
    print()
    
    total_chances = 10
    hits = 0
    
    for i in range(total_chances):
        print(f"\n第{i+1}次机会:")
        if play_once():
            hits += 1
    
    # 计算结果
    final_score = calculate_score(hits)
    hit_rate = calculate_hit_rate(hits, total_chances)
    
    # 显示结果
    print("\n=== 游戏结束 ===")
    print(f"命中次数: {hits}/{total_chances}")
    print(f"命中率: {hit_rate:.1f}%")
    print(f"最终得分: {final_score}分")

if __name__ == "__main__":
    main()
```
