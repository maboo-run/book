"现在我们来学习break和continue。"马老师说,"它们是控制循环的两个关键字。"

"break:强制跳出循环。"

```python
count = 1

while count <= 10:
    if count == 5:
        break  # 当count等于5时,跳出循环
    print(count)
    count = count + 1

print("循环结束")
```

运行:

```
1
2
3
4
循环结束
```

"看,虽然条件是count <= 10,但遇到break后,循环立刻终止,后面的6、7、8、9、10都不打印了。"

"continue:跳过本次循环,直接进入下一次。"

```python
count = 0

while count < 10:
    count = count + 1
    if count % 2 == 0:
        continue  # 如果是偶数,跳过本次循环
    print(count)

print("循环结束")
```

运行:

```
1
3
5
7
9
循环结束
```

"看,偶数都被跳过了,只打印奇数。"

"打个比方:你在跑马拉松。"

        break
